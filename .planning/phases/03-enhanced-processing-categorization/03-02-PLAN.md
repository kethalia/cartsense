**1. Create `src/lib/utils/image-enhance.ts`:**

Using sharp (already in dependencies — used in `src/lib/utils/image.ts`):

```typescript
export async function enhanceReceiptImage(base64: string): Promise<string>
```

Enhancement pipeline for receipt images:
- Decode base64 → sharp buffer
- **Auto-level normalization:** `sharp.normalize()` — expands dynamic range for faded receipts
- **Slight sharpening:** `sharp.sharpen({ sigma: 1.0 })` — improves text edges
- **Contrast boost:** `sharp.linear(1.15, -(128 * 0.15))` — subtle contrast increase
- **Grayscale conversion:** `sharp.grayscale()` — removes color noise, focuses on text (receipt text is always black/dark)
- Output as JPEG (quality 95 to preserve text detail)
- Return base64 string

Do NOT resize — the image is already compressed to max 4.5MB by the capture flow.

Add a config constant in `src/lib/config.ts`:
```typescript
export const IMAGE_ENHANCE_ENABLED = true
```

**2. Integrate into `src/lib/actions/extract-receipt.ts`:**

Before calling `extractReceiptData()`, enhance the image:

import { enhanceReceiptImage } from '@/lib/utils/image-enhance'
