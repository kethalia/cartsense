**1. Create `src/lib/actions/receipt-list.ts`** тАФ server action to fetch receipts with category data:

```typescript
export const getReceiptList = authActionClient.action(async ({ ctx: { userId } }) => {
  const receipts = await prisma.capturedReceipt.findMany({
    where: { userId },
    orderBy: { capturedAt: 'desc' },
    select: {
      id: true,
      vendorName: true,
      totalAmount: true,
      receiptDate: true,
      capturedAt: true,
      categoryId: true,
      category: { select: { id: true, name: true, nameRo: true, slug: true, color: true, icon: true } },
      imageData: true,
      mimeType: true,
    },
  });
  return receipts;
});
```

Note: This returns imageData for thumbnail display. Consider limiting to first N receipts for performance in the future.
