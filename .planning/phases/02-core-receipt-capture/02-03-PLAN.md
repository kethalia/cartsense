**1. Create `src/hooks/use-field-sources.ts`:**

Custom hook managing field source selection state and merge computation.

```typescript
import { useState, useMemo, useCallback } from 'react'
import type { FieldSources, FieldSource, ReceiptFieldName, ManualEntryData, ExtractionResult, VerifiedReceiptData, PaymentType } from '@/types/receipt'
import { RECEIPT_FIELDS } from '@/types/receipt'

type UseFieldSourcesProps = {
  aiData: ExtractionResult | null
  manualData: ManualEntryData
  defaultSource?: FieldSource  // 'ai' when AI succeeds, 'manual' when AI fails
}

// Returns:
// fieldSources: current source map
// setFieldSource: (field, source) => void — sets source, auto-deselects other
// mergedData: computed VerifiedReceiptData based on sources
// isValid: boolean — required fields (vendor, amount) have values in merged data
// errors: Record<string, string> — validation error messages
```

Logic:
- Initialize all fields to `defaultSource` (or 'ai' if AI data exists, 'manual' otherwise)
- `setFieldSource(field, source)`: updates that field's source. Since fields are exclusive (per CONTEXT.md), setting field X to 'ai' is the only operation needed — there's no "other card" to deselect because each field can only be 'manual' OR 'ai'
- `mergedData`: computed via `useMemo`. For each field, pick from aiData or manualData based on fieldSources[field]. Parse manualData string values to numbers where needed.
- `isValid`: vendorName is non-empty AND totalAmount > 0 in merged data
- `errors`: if vendorName empty in merged → "Vendor name is required". If totalAmount ≤ 0 in merged → "Total amount is required"
